Etch-A-SketchThe Odin Project

Live Demo|

This is a small project which part of the web development Foundations course within The Odin Project curriculum. This project aims to develop JS DOM manipulation skills via the creation of an Etch-A-Sketch App.

As shown below the app is capable of allowing the user to colour in their choice of colour or in a randomised rainbow mode aswell as erase what they did via hovering the mouse over the white canvas. It also allows the user to determine the resolution of the grid or reset the whole canvas.

What did I learn?

  • Use of HTML, CSS and JS
  • Use of JS to create/remove HTML elements via DOM manipulation
  • Use of DOM manipulation to alter the properties of existing/newly created HTML items
  • Using Github for version control
  • Using Github pages to deploy applications
